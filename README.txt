
Server BACK pour une application de gestion de tâches.

codé en nodejs

Pour la formation LearningCampus



pour lancer le projet en dev : nodemon run

en mode test : npm test

en mode server de prod : npm start


BEFORE :

Il vous faut un serveur mongo


Créer un fichier .env
Ce fichier dois contenir les properties suivantes :

NODE_DEBUG = false
NODE_ENV=development

mongoUser="<VOTRE USER MONGO>"
mongoPassword="<VOTRE MOT DE PASSE MONGO"
mongoUrl="<URL DE CONNEXION MONGO>"

jwtSecret="<CLE DE CHIFFREMENT POUR LES MOTS DE PASSE>"
